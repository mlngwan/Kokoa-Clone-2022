1. vsc는 코드 오작성시 빨간색으로 표시된다.

2. atrribute 값은 항상 "" 큰따옴표 안에 작성한다.

3. id라는 attribute를 가지는 tag들이 존재한다. ex) image, paragraph, header ...

4. src라는 attribute는 모든 tag가 가질수 없다.

5. 코드 자체에 의미가 부여된 semantic tag가 있다. ex) header, navigation, footer ...

6. semantic tag로 코드를 작성하는 것은 중요하다. 보기에 훨씬 더 좋고 좋은 프로그래머가 되기 위해선 필수 사항이다. 다른 사람과 공유할때 이해도 상승!

7. header, main, footer, navigation, hgroup 등 <>속 태그들은 전부 container 이다. 전부 div tag로 대체 가능하다.

8. div tag는 가장 통용적인 container이다. 대체가능하지만 코드를 보고 의미를 파악하기 위해 semantic tag를 사용한다.

9. 모든 tag와 attribute는 암기할 필요가 없다. 필요시 검색을 통해 적용가능하다.
